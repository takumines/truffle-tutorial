# truffle tutorial
https://trufflesuite.com/tutorial/

# set up
### truffle install
```
npm install -g truffle
```

### install Ganache
https://trufflesuite.com/ganache/  

### contract compile
```
truffle compile
```

### run migration
```
truffle migrate
```

### run test
```
truffle test
```

### run server
```
npm run dev
```
